# 百日安全 · 专项活动页

“安全生产·百日专项活动”的网页版工具，供班组 13 人日常学习、考试与刷题用。

**在线访问：<https://dinishabaobie.github.io/bairi-anquan/>**（手机浏览器直接打开即可，无需安装）

- 活动周期：2026.07.20 — 10.27，共 100 天 / 15 周
- 题库：130 道 · 10 类风险（单选 60 / 多选 30 / 判断 30 / 简答 10），与《夏季百日安全分类题库》纸质版一致

## 页面功能

| 卡片 | 说明 |
| --- | --- |
| 今日任务 | 自动显示第几天、本周主题、当天要做的事（视频 / 答题 / 考试 / 讨论） |
| 每日 5 题 | 全员同一组题，答对 3 题及以上合格，出分后一键复制成绩发群里 |
| 周考 / 月考 / 结业考 | 周四周考 10 题；每月最后周四月考 20 题；第 100 天结业考 30 题，80 分合格 |
| 百炼成钢 · 全库刷题 | 130 题全库按组刷，每组 10 题、逐题即时判分；可按题型筛选，断点续刷，答错的进“错题回炉”，练对销号 |
| 错题 | 每日/周考答错的题自动汇总，重练对一次自动销号 |
| 本周简答 | 按周主题列出简答题与参考答案，班前会互相提问用 |
| 逐日安排 | 100 天完整日程，含节假日、调休、各周主题 |

## 使用说明

- 每日答题成绩**不上传服务器**，只在本机；合格与否以复制到工作群的成绩为准。
- 刷题进度存在本机浏览器里，换手机/清缓存会清零，不影响考试成绩。
- 想预览任意一天的任务，在网址后加 `?d=日期`，如 `?d=2026-08-03`。

## 维护

整站就一个 `index.html`，零依赖、深浅色自适应。两处可配置项在文件开头：

- `NAMES`：填入 13 人名单后，姓名栏变成下拉选择
- `VIDEO_LINKS`：各周警示视频链接，如 `{1:"https://..."}`

改完推送 `main` 分支，GitHub Pages 自动发布。
